The Hospital Price Transparency Initiative is the aggregation of hospital service charges for every major hospital in North Carolina. These charges include inpatient, outpatient, and drug charges. 

Data 

Understanding the Data

We have profiled all of the data in each of the layers from raw to curated to presentation. The raw data is outputted as unformatted data per hospital. The curated layer is grouped by each health system found and structured to represent the max negotiated rates by payer. The presentation layer contains the grouped curated data for all systems.
